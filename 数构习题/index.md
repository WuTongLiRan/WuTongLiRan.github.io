5.试分析下列各算法的时间复杂度。

```c
x = 90;
y = 100;

while (y > 0) {
    if (x > 100) {
        x = x - 10;
        y--;
    } else
        x++;
}
```

> $O(1)$

```c++
for (i = 0; i < n; i++)
    for (j = 0; j < m; j++)
        a[i][j] = 0;
```

> $O(n×m)$

```c++
s = 0;
for (i = 0; i < n; i++)
    for (j = 0; j < n; j++)
        s += B[i][j];
sum = s;
```

> $O(n^2)$

```c++
i = 1;
while (i <= n)
    i = i * 3;
```

> $O(log_3n):3^{f(n)}\leq n$

```c++
x = 0;
for (i = 1; i < n; i++)
    for (j = 1; j <= n - i; j++)
        x++;
```

> $O(n^2):n-1+n-2+...+1=n(n-1)/2$

```c
x = n; // n > 1
y = 0;
while (x > (y + 1) * (y + 1)) {
    y++;
}
```

> $\mathrm{O}(\sqrt{n}):x\geqslant(f(n)+1)^2,\text{又由于}x=n,\text{ 所以 }f(n)\leqslant\sqrt{n}-1$
