关键词大小写绕过

有的WAF因为规则设计的问题，只匹配纯大写或纯小写的字符，对字符大小写混写直接无视，这时，我们可以利用一点来进行绕过

1

union select ---> unIOn SeLEcT

编码绕过

针对WAF过滤的字符编码，如使用URL编码，Unicode编码，十六进制编码，Hex编码等

1

union select 1,2,3# ---> union%0aselect 1\u002c2,3%23
