魔术引号

魔术引号（Magic Quote）是一个自动将进入 PHP 脚本的数据进行转义的过程。

当打开时，所有的 ‘（单引号），"（双引号），\（反斜线）都会被自动加上一个反斜线进行转义。这和 addslashes() 作用完全相同。
