文档整理 by七折（tccc）

登录

**提交接口:**http://floor.huluxia.com/user/info/ANDROID/4.1.8

需提交参数:

  • platform=2
  • app_version=APP版本[建议最新版][!]
  • gkey=000000
  • market_id=登录应用 [floor_web葫芦侠三楼,floor_tool][更改请自换接口]
  • floor_webversioncode=20141481
  • _key=
  • phone_brand_type=手机品牌类型[!]
  • account=账号[!]
  • login_type=登录类型[!][推荐2]
  • password=密码[请用md5加密]
  • sign=符号?
